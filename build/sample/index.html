Export Anki APKG deck.

Anki deck generation based on https://github.com/krmanik/genanki-js
Call this page with encoded URI params as follows: 
    .....?data=[
                {front:"q1", back:"a1"},
                {front:"q2", back:"a2"},
                ...
               ]
e.g. 
       ?data=[{"front"%3A"q1"%2C"back"%3A"a1"}%2C{"front"%3A"q2"%2C"back"%3A"a2"}]
If this page has been called correctly your browser should download a file called "Deck*.apkg" and offer it to be opened by Anki on a mobile device.
