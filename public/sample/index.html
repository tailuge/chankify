Exporting Anki APKG deck.



Anki deck generation based on https://github.com/krmanik/genanki-js
Call this page with encoded URI params as follows: 
    .....?data=[
                {q:"q1", a:"a1"},
                {q:"q2", a:"a2"},
                ...
               ]
e.g. 
       ?data=[{"q"%3A"q1"%2C"a"%3A"a1"}%2C{"q"%3A"q2"%2C"a"%3A"a2"}]
alternatively large amounts of data can be passed in session storrage URI encoded with location passed as ?sessionId=...
